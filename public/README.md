# Public Assets

Per aggiungere lo sfondo personalizzato per la pagina Profilazione:

1. Salva l'immagine come `sfondo-dota.jpg` in questa cartella (`public/`)
2. L'immagine verrà automaticamente utilizzata come sfondo nella pagina Profilazione FZTH
3. Se l'immagine non è disponibile, verrà utilizzato un gradiente di fallback

Formati supportati: .jpg, .png, .webp
